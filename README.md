# PicHub (本地智能相册)

## 快速开始

### 运行环境要求
*   开发工具：Android Studio
*   最低支持系统：Android 7.0 (API Level 24)

### 下载与安装

#### 1. 直接下载安装包（推荐给普通用户）
无需任何开发环境，如果你只是想安装使用这款相册：
1. 打开本仓库右侧的 **[Releases]** 页面。
2. 找到最新版本的发布记录（Latest）。
3. 在 **Assets** 下方点击下载 `.apk` 安装包即可直接安装。

#### 2. 开发者自行编译
如果你是开发者，并且想要修改代码，可以直接在 Android Studio 中点击 Run 按钮运行项目，或者在终端执行以下命令进行构建：

```bash
./gradlew assembleDebug
```
